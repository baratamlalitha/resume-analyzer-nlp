# 📄 AI Resume Analyzer

An end-to-end resume analyzer that extracts text from resumes, performs basic NLP preprocessing, matches resumes with job descriptions using TF-IDF and cosine similarity, identifies missing skills, and generates a structured learning roadmap using a Streamlit interface.

---

## 🚀 Features

- 📄 Resume text extraction (PDF)
- 🧠 NLP-based text preprocessing
- 📊 Resume-job similarity scoring (TF-IDF + Cosine Similarity)
- ❌ Missing skills identification
- 📍 Learning roadmap generation
- 🌐 Streamlit web interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- PDFPlumber
- Regular Expressions (re)
